# SmartStock AI

SmartStock AI is an AI-powered Inventory Management and Demand Forecasting System.

## Tech Stack

- React.js
- Node.js
- Express.js
- MySQL
- Python (FastAPI)
- Scikit-learn

## Features

- Authentication
- Product Management
- Inventory Tracking
- Sales Management
- Demand Forecasting
- Low Stock Alerts
- Dashboard Analytics
